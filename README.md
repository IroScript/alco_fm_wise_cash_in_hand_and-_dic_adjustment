# Alco Field Manager Cash In Hand Formatting

A automated Python script to split, structure, and format hierarchical Cash in Hand MPO and DA sales daily tracker files on a per-FM (Field Manager) basis.

## Features
- **AD 50,000 Futuristic UI Theme:** Beautiful dark void backgrounds, neon cyan titles, midnight slate headers, light zebra rows, and mint green total summary columns.
- **Auto-Calculated Summary Formulas:** Generates daily `=SUM(...)` formulas for each date of June 2026.
- **Vacancy Detection & Hiding:** Automatically flags vacant positions and hides their columns.
- **Metadata Protection:** Hides metadata rows (Zone, FM, Market, MPO Code, FM Code, Designation, DA name) to maintain a minimal, professional aesthetic upon opening, while keeping the data accessible for unhiding.
- **Window Freezing & Zoom:** Freezes headers (name + date) and sets the zoom level to 90% for a optimal overview in Excel and Google Sheets.
